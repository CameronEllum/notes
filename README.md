# notes
Command line music tagger

Tags MP3 and AAC files

Help:

Allowed options:
  --help                    produce help message
  -s [ --strip ]            Strip existing tags
  -t [ --title ] arg        Title
  -n [ --track ] arg        Track
  -p [ --performers ] arg   Performer(s)
  -c [ --composers ] arg    Composer(s)
  -g [ --genre ] arg        Genre
  -y [ --year ] arg         Year
  -d [ --disc ] arg         Disc
  -a [ --album-title ] arg  Album title
  -r [ --album-artist ] arg Album artist
  --art arg                 Art (cover)
  -i [ --input-file ] arg   Input file
